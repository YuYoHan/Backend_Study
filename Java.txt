JAVA

자바란?
    - 컴파일 언어
    - 객체 지향 언어

JVM(자바 가상 머신)
    - 자바로 만들어진 프로그램을 실행하는 'os'
    - 플랫폼 독자적인 실행 환경을 제공
    - 자동 메모리 관리(GC)를 수행

JRE(자바 런터암 환경)
    - JVM을 실행하기 위한 자바 실행 환경
    - 자바 실행에 필요한 장비들을 가지고 있다.

JDK(자바 개발 킷)
    - 자바 프로그램을 만들기 위한 개발자용 개발 도구
    - 자바 개발에 필요한 라이브러리을 가지고 있다.

↑ 위 3개는 JAVA가 돌아가는데 필요한 환경

컴파일
    사람 언어를 컴퓨터 언어로 번역

컴파일러
    자바는 JAVAC로 컴파일한다.
